# AML Transaction Monitoring Dashboard | Power BI

## Overview
This project focuses on analyzing banking transactions to identify suspicious patterns and risk indicators using Power BI.

## Objective
- Monitor transaction behavior
- Identify high-risk transactions
- Analyze customer risk profiles
- Visualize AML compliance indicators

## Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## Dashboard Features
- Total Transactions
- Transaction Value Analysis
- Risk Category Distribution
- Suspicious Transaction Monitoring
- Customer Risk Analysis

## Dataset
The dataset contains banking transaction records including:
- Transaction ID
- Customer ID
- Transaction Amount
- Country
- Transaction Type
- Risk Score
